# SlopFactory Official Website

The official landing page for SlopFactory, an independent AI-native software
studio building custom web, mobile, and intelligent systems.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Update public contact details in `src/siteConfig.js` when they are ready.

Pushes to `main` are prepared to deploy through GitHub Pages.
